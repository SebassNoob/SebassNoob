About this site


This page was originally created to give my html skills a whirl. However, as the project progressed, I realised that there wasn't a clearly defined purpose of this page. Therefore I decided to dedicate this page to showcasing myself and my skills, transforming it into an online e-portfolio that I could show to others.

Technical Info


Backend systems:
    • Python with Flask ('pip install flask')
      Hosted by replit.com, an online IDE offering free hosting services.
    • Frontend UI:

    • Barba.js, a lightweight page animations helper
      jinja2, a templating engine in html (a flask dependency). Special placeholders in the template allow writing code similar to Python syntax.

    • Testing and QA done on Safari, Chrome, and Microsoft Edge. View page speed insights on PageSpeed
